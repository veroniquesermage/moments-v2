== Architecture modulaire - Backend

....
backend/
├── app/
│   ├── core/
│   │   ├── auth/              # Authentification (OAuth, JWT, refresh tokens)
│   │   ├── users/             # Gestion utilisateurs + comptes tiers
│   │   └── common/            # Utilitaires, exceptions, config
│   │
│   ├── modules/
│   │   ├── gifts/             # Cadeaux (CRUD, réservations, partage)
│   │   ├── recipes/           # Carnet de recettes (ingrédients, conversions pint)
│   │   ├── menus/             # Planification menus (utilise recipes)
│   │   ├── accommodation/     # Organisation maison/chambres
│   │   ├── table_plan/        # Plans de table (algo protocole français/anglais)
│   │   ├──contacts/          # Gestion contacts/amis
|   |   └── program/
|   |       ├── domain/
|   |       |   ├── activity.py
|   |       |   └── participant.py
|   |       ├── application/
|   |       |   ├── create_activity.py
|   |       |   ├── program_reader.py
|   |       |   └── participant_provider.py
|   |       ├── infra/
|   |       |   ├── activity_repository.py
|   |       |   ├── activity_entity.py
|   |       |   └── person_adapter.py
|   |       └── presentation/
|   |           ├── activity_routes.py
|   |           └── activity_schemas.py
│   │
│   ├── contexts/
│   │   ├── personal/          # Espace perso (profil, liste amis)
│   │   ├── events/            # Événements (création, templates, orchestration)
│   │   └── dinners/           # Organisation dîners (historique)
│   │
│   ├── tasks/                 # Celery tasks (cleanup J+7, expirations)
│   └── storage/               # Gestion fichiers (abstraction VPS)
│
├── tests/
│   ├── unit/
│   └── integration/
│
├── alembic/                   # Migrations base de données
│   └── versions/
│
├── requirements.txt
├── .env.example
└── main.py                    # Point d'entrée FastAPI
....
