= Réflexion sur le modèle de données Moments

== Principes retenus

Nous distinguons trois concepts :

[source,text]
----
Account
= compte technique permettant de se connecter

Person
= personne métier connue de Moments

EventParticipant
= participation d'une Person à un Event donné
----

== Modèle conceptuel

[source,text]
----
Account
  |
  v
Person
  |
  v
EventParticipant
  |
  v
Event
----
